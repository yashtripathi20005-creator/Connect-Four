# Connect Four

A classic two-player Connect Four game built with Python and Pygame.

## Description

Connect Four is a two-player connection game in which players take turns dropping colored discs into a vertical, suspended grid. The objective is to be the first to form a horizontal, vertical, or diagonal line of four of one's own discs.

## Features

- Two-player gameplay (local multiplayer)
- Visual game board with colored pieces
- Win detection for all four directions
- Draw detection
- New game button
- Turn indicator

## Requirements

- Python 3.6 or higher
- Pygame 2.5.2
